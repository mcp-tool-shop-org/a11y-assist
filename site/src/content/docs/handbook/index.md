Welcome to the a11y-assist handbook. This is the complete guide to adding structured, accessible recovery plans to CLI failures.

a11y-assist is deterministic, additive, and SAFE-only. It never rewrites tool output -- it only adds a structured ASSIST block. Five accessibility profiles ship out of the box, and confidence is always disclosed.
